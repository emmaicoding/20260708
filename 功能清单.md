## 四、通用交互规则

```
┌─────────────────────────────────────────────────┐
│  输入校验                                        │
│  ├─ readNonEmpty  → 空值重新提示                 │
│  ├─ readInt(min,max) → 超范围重新提示            │
│  ├─ readDate      → 格式校验，返回LocalDate      │
│  ├─ readPhone     → 11位手机号校验               │
│  └─ readGender    → 仅接受 M/F（不区分大小写）   │
│                                                  │
│  确认操作                                        │
│  └─ readConfirm   → (Y/N)，大小写均接受          │
│                                                  │
│  导航                                            │
│  ├─ 0 = 返回上级菜单                             │
│  └─ waitForEnter → 按回车键继续...               │
│                                                  │
│  数据安全                                        │
│  ├─ 幼儿删除 = 逻辑删除（status标记离园）        │
│  ├─ 调班 = 事务处理（原子更新班级人数）          │
│  └─ 密码 = SHA-256哈希存储                       │
└─────────────────────────────────────────────────┘
```

# 更新日志

本项目遵循 [语义化版本控制](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2026-07-08

### 新增
- 幼儿学籍管理（增删改查、按班级查看、按姓名搜索）
- 课程管理（增删课程、选课退课、容量校验）
- 食谱管理（菜品库管理、每周排餐、复制上周食谱）
- 调班管理（跨年级调班、事务控制、调班记录）
- 考勤管理（逐个/批量打卡、历史查询、出勤率统计）
- 数据统计（班级人数、课程选课、年级分布、出勤率）
- 管理员和教师双角色权限控制
- 数据库自动初始化（建库建表 + 预置数据）
- 密码 SHA-256 哈希存储
- DAO 层统一异常处理（DataAccessException）
- 单元测试（8 个测试类）

### 技术栈
- Java 8+
- MySQL 8.x
- JDBC 直连（无 ORM 框架）
- 分层架构（View → Service → DAO → Entity）
